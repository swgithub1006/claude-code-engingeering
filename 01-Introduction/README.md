# Claude Code 工程化实战

## 课程简介

本教程系统性地介绍 Claude Code 的高级功能和工程化实践。帮助你从"会用 Claude Code"提升到"精通 Claude Code"做项目。

## 适合人群

- 已有 Claude Code 基础使用经验的开发者
- 希望提升 AI 辅助开发效率的工程师
- 想要构建自定义 AI 工作流的团队

## 章节概览

| 章节 | 主题 | 核心内容 |
|------|------|----------|
| 02 | Memory | CLAUDE.md 记忆系统 |
| 03 | SubAgents | 子代理与任务分发 |
| 04 | Plugins | 插件开发与分发 |
| 05 | Commands | 自定义斜杠命令 |

## 学习路径

### 路径 A：快速入门（1-2天）
```
02-Memory → 05-Commands → 03-SubAgents
```
掌握基础配置和日常使用优化。

### 路径 B：团队协作（3-5天）
```
02-Memory → 04-Plugins → 05-Commands → 06-Skills → 07-Hooks
```
构建团队共享的工作流和规范。

### 路径 C：完整进阶（1-2周）
```
按顺序学习所有章节
```
全面掌握 Claude Code 的工程化能力。

### 路径 D：自动化专精
```
07-Hooks → 09-Headless → 10-Agent-SDK
```
专注于 CI/CD 集成和自动化场景。

## 开始学习

1. **克隆/下载**本教程
2. 从 `02-Memory` 开始，了解基础配置
3. 根据你的需求选择学习路径
4. 每章先看课程文档，再看 `projects/` 示例
5. 动手实践，修改示例适配你的项目

## 环境要求

- Claude Code CLI（最新版本）
- Node.js 18+（部分示例需要）
- Python 3.10+（Agent SDK 示例需要）
- Anthropic API Key
